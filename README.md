# EV-Smart-Tractor
✔ Overview

Fully electric tractor designed for sustainable farming

Replaces diesel engine with a clean battery-powered drive system

Low maintenance, low noise, and zero emissions

✔ Key Features

Battery → Motor Controller → DC Motor power flow

High torque for agricultural operations

Eco-friendly design that cuts fuel cost

Simple and efficient drivetrain

Reduced vibration and noise

✔ Components Used

Electric DC Motor

Motor Controller

Rechargeable Battery Pack

Chassis Frame

Transmission/Drive system

✔ Working Principle

Battery supplies DC power

Controller regulates motor speed and torque

Motor drives wheels

Tractor performs field tasks like ploughing and load carrying

✔ Advantages

Zero pollution

No fuel cost

Lower maintenance

High energy efficiency

✔ Applications

Ploughing

Transporting loads

Small-farm field operations
